# expect-mcp

Custom [Vitest](https://vitest.dev/) matchers to write automated test assertions for Model Context Protocol (MCP) tools.

# Installation

```
npm i expect-mcp
```

See the [Docs page](https://facetlayer.github.io/expect-mcp/) for more info.
